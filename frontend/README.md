   # TODO List (MERN Stack)

This is a full-stack TODO List application built using:

- MongoDB
- Express.js
- React.js
- Node.js

## ✨ Features

- ✅ Create Task
- 📝 Update Task
- ❌ Delete Task
- 🔍 Search Task (by name, date, status)
- ⏰ Reminder Functionality
- 📅 Filter/Sort by Date or Name
